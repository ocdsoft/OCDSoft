Software Developer

I started in software development 19 years ago doing VB6 building Win apps. I've been doing ASP.Net in C# for about the last 14 years for my day job. In recent years I've enjoyed playing with Ionic, Angular, Firebase and other frameworks built on Node and JavaScript. I'm currently really excited about Blazor, which is C# in browser via WebAssembly. I'm experimenting with a static Blazor WebAssembly app and using Azure table storage as an inexpensive nosql database. I've also been getting more active on GitHub and recently submitted my first pull request to contribute to an open source Blazor library.

I live in Hermiston, OR with my amazing wife and two of three kids.
